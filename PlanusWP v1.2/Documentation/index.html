Installation

You'll want to install this theme just like any other WordPress theme, either by uploading the main theme folder via FTP (unzipped) or the zipped file via the Wordpress theme upload section

1. FTP method

  • Log into your site via FTP
  • Browse to your folder located at wp-content/themes
  • Upload the MAIN theme folder. This is the folder named "Planus" located inside the "upload" folder from the zip file.
  • Log Into Wordpress and go to Appearance --> Themes
  • Find the theme and click to activate

2. WordPress Method

  • Log into your WP dashboard
  • Browse to Appearance --> Themes
  • Click to "add new" and then follow the instructions on the screen
  • When it asks to select a file find the zipped theme folder located in the "upload" folder in the zip downloaded.

Activate and Save permalinks

Go to your Settings -> Permalink and hit the save button to prevent any 404 errors on custom post type pages. If you can handle it, you can change the permalinks to custom structure by choosing the "Custom structure" choice and insert "/%category%/%postname%/" into that field. This will change the permalinks to category-based structure.

Hit Save and follow the instructions regarding the .htaccess file.

Home page Settings

In order to set the base layout of the Planus theme, you need to follow the next instructions:

  1. Create a new page named Home. Be shure to choose the appropiate template (named Home Page Template) from Page Attributes section on the right side. Also be shure to press the "Publish" button.
  2. Create a new page named Blog using the same techinque as for Home, but this time choose the Blog template
  3. Go to Settings -> Reading and choose to show "A static page" as home. Select the Home page for Front page and Blog for Post page. Hit "Save Changes" button
  4. Go to Appearance -> Theme Options -> General Settings tab and fill all fields accordingly. You need to follow the instructions from the right side of the screen. Press "Save Changes" button.
  5. Now you will start to add all your section on the home page. To do that, go to "Home Section" tab and press the "Add New" button. This will open a form with four fields. Please pay attention to all fields:
    • Title - you can add any title you want. This is not the title of the section.
    • Section title on menu - This is the text that appears on the top menu from the home page. If you leave it empty, this section will be omitted from the menu.
    • Choose Predefined Section - this field will shou you all predefined sections. You can change the order of your sections after
    • Select your custom page (from v1.2) - If you don't want to add a predefined section, maybe you need to add a custom one. You just need to create a page first and then select it on this field. If you select this field, the previous one (predefined sections) will be omitted.
  6. Repeat the last step for each section you want to display.
  7. Right now, all sections are visible on your home page but we are not finished. We need to set up each section and add your informations on them.

    8. For you information - You can change the order of home page sections eith drag-and-drop the elements created by you on theme options.

Welcome Screen

  • Go to Appearance -> Theme Options -> Welcome Section tab and fill all fields accordingly. You need to follow the instructions from the right side of the screen. Press "Save Changes" button.
  • Don't worry if you don't want to fill all the fields this time.

About Section

  • Go to Appearance -> Theme Options -> About Section tab and fill the fields like this:
    • Title - this is the title shown on the top of the About Section
    • Title color - this will change the color of the About section title
    • Content color - The color for content. This is very useful when you want to upload a dark background and the text becomes unreadable.
    • Background Settings - You can set your background according to your needs
    • Upload images - Just some images with you or your company. You can live it blank if you want but keep in mind that you can upload only 3 images.
    • About Content - the About Me content
    • Social Icons - press the "Add New" button to add social icons on the About Me section. After adding them all, you can change the order as you wish.

Portfolio Section

  1. Go to Portfolio Items -> Portfolio Categories and create some categories for your portfolio. These categories will be shown on the home page as filter buttons for portfolio.
  2. Go to Portfolio Items -> Add New and add some portfolio posts. For each portfolio post you have 2 options. If you want to have more images as a slider you need to use the Gallery Images section. By pressing "Add New" button you can add as much images as you need. Another option is to set the featured image and this is the case when you have only one image for the portfolio post (the slider will be disabled). In both cases I recommend adding a featured image.
  3. Go to Appearance -> Theme Options -> Portfolio Section tab and fill the title and all fields accordingly. For each field you'll see the instructions on the right side.

Testimonials Section

For testimonials I added a new post type (Testimonials) and new taxonomy (Testimonials Category). If you don't need the categories for this section, just omit them. The testimonials will work without categories. Please follow these steps:

  1. Go to Testimonials -> Add New and start creating one testimonial. You need to fill each field like bellow:
    • Title (post title) - is required. Though, will not be displayed.
    • Content - the content of the testimonial.
    • Testimonial Details - Photo (108x180px, name of the person (John Doe) and his title (e.g. CEO) will be shown
  2. Go to Appearance -> Theme Options -> Testimonials Section tab
    • Title - The title for Testimonials section
    • Title color - this will change the color of the Testimonials section title
    • Content color - The color for content. This is very useful when you want to upload a dark background and the text becomes unreadable.
    • Background Settings - You can set your background according to your needs
    • Do not forget to save the changes

Blog Section

To set up the blog:

  1. Go to Appearance -> Theme Options -> Blog Section tab. Fill each field accordingly and on the field named "Select Blog Page" please select the "Blog" page created before.
  2. You can start adding posts by going to Posts -> Add New. Be shure to add a featured image to each post created. This time you can create and use both categories and tags.
  3. You have fields for Title color, Background and button text. please fill all fields accordingly.

Contact Section

The contact section contains some contact informations along with a nice form. The form is not using a plugin, is a stand-alone form so you only need to add you email to it. To do that, please follow the next instructions:

  1. Go to Appearance -> Theme Options -> Contact Section tab.
  2. Add the section title by filling the "Title for Contact Section"
  3. Title color - this will change the color of the Contact section title
  4. Submit button color - this will change the color of Submit button
  5. Content color - The color for content. This is very useful when you want to upload a dark background and the text becomes unreadable.
  6. Submit button color - this will change the color of Submit button
  7. Background Settings - You can set your background according to your needs
  8. Content - Add fiew words for your visitors
  9. Your email adress - this is the email where you will receive your messages
  10. Email subject - the subject of the messages sent by your visitors
  11. Adress - your place
  12. Phone Number where you can be contacted.

Menus

Home Page Menu

About the menus you need to know that the menu from the home page is automatically generated when you create the home page. Also the menu will change when you change the order of the home page sections

Starting with version 1.1, you can add more links to the top menu on home page. In order to do that, you need to create a new menu (Appearance -> Menus), give it a name, put the desired links inside and Save it. After saving you need to set a location to this menu. To show these links on the top bar of the home page, you need to check the "Home Page Menu" on Theme location section. That's it.

Inner Pages Menu

The menu you'll be adding next will be visible only on inner pages so keep that in mind.

  1. Go to Appearance -> Menus.
  2. Create a new menu
  3. Add menu elements from the left side and put them in your preferred order
  4. Add a menu name on the top
  5. Press "Create Menu" button
  6. After page refresh, check the theme location named "Inner pages Menu" and hit "Save Menu"

Shortcodes

Font Awesome Shortcode

For more informations about the icons please visit the Font Awesome

[icon type="" size="" rotate="" flip="" pull="" spin=""]

Attributes:

  • type - this is the icon from the Font Awesome collection. Please add only the name (e.g. desktop, rss)
  • size - values: lg, 2x, 3x, 4x, 5x
  • rotate - values: 90, 180, 270
  • flip - values: horizontal, vertical
  • pull - right, left
  • spin - true, false. Default is false.

Buttons Shortcode

This shortcode will help you to add buttons anywhere on your content.

[button size="" corners="" color="" text="" href=""]

Attributes:

  • size - values: small, big
  • corners - values: round, square
  • color - values: red, blue, green, white
  • text - value: Insert the text for your button
  • href - value: Insert the url for your button.

Columns Shortcodes

[row]
	[col columns="8" offset="2" breakpoint="md"]
		Here is the column content
	[/col]
[/row]

Attributes:

  • columns - number of 1/12 colmns. As you can see, you'll have 12 choices (numbers between 1 and 12). The total number of columns from one row can't be bigger than 12. In our case we have 8 columns.
  • offset - defines how many columns we leave free to the left of main columns. In our case, because we want to center our 8col content, we'll set the offset to 2.
  • breakpoint - defines the css breakpoint when the columns gets stacked. You have 3 choices: sm, md and lg (small, medium and large)

Pretty Columns Shortcode

[row]
	[prettycol icon="desktop" title="Web design" anchor="" columns="3" offset="" breakpoint=""]
		Here is the Prettycol content
	[/prettycol]
[/row]

Attributes:

  • icon - this is the icon from Font Awesome collection. You need to add only the name of the icon, not the entire class. So if you need the icon "fa-desktop", then you will add only "desktop". You can use any icons from here: Font Awesome Cheatsheet
  • title - the title for the pretty column
  • anchor - In case you want to link to a service page, add the link to it.
  • columns - number of 1/12 colmns. As you can see, you'll have 12 choices (numbers between 1 and 12). The total number of columns from one row can't be bigger than 12. In our case we have 8 columns.
  • offset - defines how many columns we leave free to the left of main columns. In our case, because we want to center our 8col content, we'll set the offset to 2. The default values is empty.
  • breakpoint - defines the css breakpoint when the columns gets stacked. You have 3 choices: sm, md and lg (small, medium and large). The default value is "md".

Tabs Shortcode - since v1.2

[tabs type="tabs"]
	[tab title="Tab1" active="true" fade="true"]
		Here goes the content for the first tab.
	[/tab]
	[tab title="Tab2" fade="true"]
		Here goes the content for the second tab.
	[/tab]
[/tabs]

Attributes:

  • type - Options: tabs, pills. This will set the type of tabs.
  • title - The title of tab. This is an option for each Tab.
  • active - Options: true, false. This will set the active tab. By default the first tab is active.
  • fade - Options: true, false. This will add the fade effect tab content. By default is set to true.

Accordion Shortcode - since v1.2

[collapsibles]
	[collapse title="Item 1" type="default" active="true"]
		Here goes the content for the first accordion item.
	[/collapse]
	[collapse title="Item 2" type="default"]
		Here goes the content for the second accordion item.
	[/collapse]
	[collapse title="Item 3" type="default"]
		Here goes the content for the third accordion item.
	[/collapse]
[/collapsibles]

Attributes:

  • title - The title for the accordion item. The value should be text or number.
  • type - Options: default, primary, success, info, warning, danger, link. This will set the type of the accordion item.
  • active - Options: true, false. This will set the active tab. By default the first accordion item is active.

Google Map Shortcode

[googlemap id="myMap" type="road" height="400" zoom="16" latitude="44.855499" longitude="24.867607" message="We are here" hue="#579F6A" ]

Attributes:

  • id - This is the ID of the map
  • type - You can choose one from: satellite, terrain, hybrid, roadmap. Default is roadmap.
  • height - This is the height of the map section. Note that the width is set by default to 100%
  • zoom - Zoom value for closer view.
  • latitude - The latitude of your place.
  • longitude - The longitude of your place.
  • message - Message shown when click on map pin
  • hue - By using this value you can set a hue over your map. I recommend to set the same color as Contact section.
